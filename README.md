# FUTURE_DS_02
# Customer Retention & Churn Analysis Dashboard

# Project Overview
This project analyzes telecom subscription data to understand customer churn patterns and retention behavior. The dashboard provides insights into churn rate, revenue impact, contract-based risk, tenure distribution, and payment behavior.

# Objective
- Calculate churn and retention rates
- Identify high-risk customer segments
- Analyze churn drivers (contract, tenure, payment method)
- Measure revenue loss due to churn
- Provide strategic recommendations

 #Tools Used
- Power BI Desktop
- DAX (Data Analysis Expressions)
- Power Query (Data Cleaning)
- IBM Telco Customer Churn Dataset

# Key Performance Indicators (KPIs)
- Total Customers
- Churned Customers
- Active Customers
- Churn Rate %
- Retention Rate %
- Revenue Lost Due to Churn

# Dashboard Components

# Customer Churn Distribution
Pie chart showing churned vs retained customers.

# Churn by Contract Type
Identifies which contract types have highest churn.

# Churn by Payment Method
Shows churn behavior based on payment preferences.

# Churn by Tenure Group
Analyzes early vs long-term customer churn patterns.

# Key Insights
- Month-to-month contracts have the highest churn rate.
- Early tenure customers (0â€“12 months) are most likely to churn.
- Electronic check payment users show higher churn levels.
- Churn contributes significantly to revenue loss.
- Long-term contracts improve retention stability.

# Strategic Recommendations
- Promote long-term contract plans.
- Improve onboarding experience for new customers.
- Encourage automatic payment methods.
- Implement retention campaigns for early-risk segments.
